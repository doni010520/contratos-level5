# Assets do Contrato

Coloque os seguintes arquivos nesta pasta:

## capa.png
- Imagem da primeira página do contrato (capa)
- Dimensões recomendadas: 595 x 842 pixels (A4 em 72dpi)
- Ou: 2480 x 3508 pixels (A4 em 300dpi para alta qualidade)
- A imagem deve ter espaço para o nome do cliente na parte inferior

## logo.png
- Logo da Level 5 Engenharia Elétrica
- Dimensões recomendadas: 200-400 pixels de largura
- Fundo transparente (PNG com alpha)
- Será usado no header das páginas internas do contrato

---

**Nota:** Sem esses arquivos, o PDF será gerado com layout básico de fallback.
